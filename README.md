# 1. 페이지 선택이유

- 국내 최대규모의 한정 아이템 중개 거래소
- 기존의 커머스 사이트와는 다른 판매기능 구현가능
- 구매 입찰과 판매입찰을 통한 경매시스템 구현 가능

### FE

- 김민경 : 네브, 푸터작업, 상세 페이지
